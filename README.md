## Instalacion
Necesitaras las siguiente herramientas para poder ejecutar el proyecto:
- Node.js 6.5+
- My SQL 8+
- npm 4+
- Angular cli
- Angular 8.x
- Ionic 2.x  
## Ejecutar el proyecto
Una vez instalado todos los programas necesarios, solo necesiatas clonar nuestro repositorio:

`gh repo clone <repo>`
